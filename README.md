# SonInk — AI 语音绘图工具

纯语音控制的绘图工具：用户不使用鼠标和键盘，仅通过语音指令完成绘图创作。Vue 3 + TypeScript + Konva.js，纯前端，无后端。

## 快速开始

```bash
npm install          # Node 18+
npm run dev          # 启动开发服务器 → http://localhost:5173
```

用 **Edge 浏览器** 打开（Web Speech API 在国内需走 Edge 的 Azure 语音服务；Chrome 走 Google 服务不可用）。授予麦克风权限后，直接说话即可绘图。

### 大模型配置（可选）

基础图形（圆、矩形、三角、直线、文字、自由笔刷）无需 AI Key 即可绘制。**绘制太阳、房子、汽车等任意复杂/语义图形均由 AI 拆解完成**，需点击右上角 **设置**，选择服务商并填入 API Key：

| 服务商 | 默认模型 |
|--------|----------|
| DeepSeek | `deepseek-chat` |
| Kimi（月之暗面） | `moonshot-v1-8k` |
| 智谱 GLM | `glm-4-flash` |
| 通义千问 | `qwen-plus` |
| 自定义（OpenAI 兼容） | 自行填写 |

Key 仅存浏览器 localStorage，不进代码仓库、不上传至所选大模型以外的任何服务。

## 使用说明

### 语音指令示例

#### 基础绘图
| 指令 | 效果 |
|------|------|
| "画一个红色的圆" | 画布中央画红色圆形 |
| "在左上角画一个大三角形" | 左上角画大三角形 |
| "画一个蓝色的方块" | 画蓝色矩形 |
| "从这里到这里画一条线" | 画直线 |

#### 组合图形（由 AI 拆解，需配置 Key）
| 指令 | 效果 |
|------|------|
| "画一个太阳" | 一笔描画太阳轮廓（path） |
| "画一座房子" | 一笔描画房子 |
| "画一棵树" | 一笔描画树 |
| "画一个雪人" | 一笔描画雪人 |
| "画一个小人" | 一笔描画小人 |
| "画一个笑脸" | 一笔描画笑脸 |

#### 选中与编辑
| 指令 | 效果 |
|------|------|
| "选中红色的圆" | 高亮匹配对象 |
| "把它往右移一点" | 移动选中对象 |
| "放大一倍" | 缩放选中对象 |
| "把这个圆变成蓝色" | 改色 |
| "删掉这个圆" | 删除 |

#### 复合指令与高级操作
| 指令 | 效果 |
|------|------|
| "画一个圆，然后画一个方形" | 连接词拆分，顺序执行 |
| "在圆的右边画一个三角形" | 相对定位（左/右/上/下） |
| "在圆和三角之间画方形" | 双锚点之间定位（between） |
| "左边那个放大" | 空间指代（最左/最右/最大/第二个） |
| "把它放大" / "小一点" | 缩放（含文字） |
| "画一辆小汽车" | LLM 拆解为多部件 path（需配 Key） |
| "写上'你好'" | 文字标注 |
| "清空画布" → "确认" | 清空（需二次确认） |
| "撤销" / "重做" | 撤销/重做 |
| "保存图片" | 导出 PNG |
| "回放操作" | 历史回放 |
| "天空背景变为蓝色" | 设置背景色 |

#### 自由笔刷
| 指令 | 效果 |
|------|------|
| "开始画线" | 进入笔刷模式，画布中心出现起点 |
| "往右" / "往下" | 笔触向右/下延伸一步 |
| "往右上" | 斜向延伸 |
| "往下移很多" | 大步长延伸 |
| "停" | 结束笔刷，路径定型为可编辑线条 |
| "取消" | 放弃当前笔画 |

笔刷模式下每个方向指令立即更新画布（实时预览），定型后自动播放逐笔描画动画。

#### 鼠标拖拽

可直接用鼠标拖拽画布上的图形调整位置，拖拽操作计入撤销历史，之后可继续语音控制。

### 键盘快捷键

| 快捷键 | 操作 |
|--------|------|
| Ctrl+Z | 撤销 |
| Ctrl+Shift+Z / Ctrl+Y | 重做 |

### 实时反馈

- **字幕条**：顶部实时显示识别中的文字（灰色）和确认后的最终文字（黑色）
- **TTS 播报**：语音播报执行结果、追问、确认提示（播报期间自动暂停麦克风，防止自听回环）
- **画布高亮**：选中对象显示高亮框
- **操作历史面板**：右侧滑出面板，显示已执行指令列表

## 架构概览

```
麦克风 → Web Speech API → pipeline.ts → parser/rules.ts（快路径，~50ms）
                                        ↘ llm/client.ts（慢路径，~1-2s）
                                                 ↓
                                            dsl/schema.ts（校验）
                                                 ↓
                                            dsl/executor.ts → Konva 画布
```

- **双通道解析**：简单指令走本地规则引擎（正则 + 同义词表），毫秒级响应；复合指令走 LLM 拆解
- **统一 DSL**：所有指令解析为同一 JSON 格式，执行引擎只认 DSL
- **事务快照撤销**：整体状态快照而非逐指令逆操作，复合指令一次撤销整体回滚
- **path 组合**：复杂图形用连续坐标点一笔描画，支持 groupName/part 细粒度指代
