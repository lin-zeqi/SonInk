# 开发记录 #06 — 选中 / 移动 / 删除与指代消解（P0 范围）

- **日期**：2026-06-12
- **分支**：`feat/06-select-move-delete`
- **对应路线图**：PR #6

## 做了什么

1. **DSL 扩展**（types.ts + schema.ts）：新增 `select` / `move` / `delete` 三个 action 与
   `TargetSpec` 目标描述结构（`ref: last|selected` + 特征 shape/color）。move 支持
   相对（方向+语义/绝对步长）与绝对（九宫格位置）两种模式，schema 校验同步覆盖。
2. **指代消解 `resolveTarget`**（executor.ts，P0 收窄范围，design.md §3.3）：
   - 空 target → 当前选中，否则最近对象；
   - `ref: last` → 最近创建的对象；`ref: selected` → 当前选中（无选中退化为最近）；
   - 特征匹配 → shape/color 过滤全部对象；**有选中且与匹配集有交集时优先取交集**
     （"选中红圆"之后说"删掉这个圆"只删选中的那一个，不误伤其他圆）；
   - `last` + 特征同时存在 → 匹配集中最近的一个（"刚才那个圆"）。
3. **多候选不猜测**：select 匹配多个 → 全部高亮 + 反馈"请说得更具体些"；
   move 匹配多个 → 拒绝执行并提示先选中。
4. **选中高亮**：反馈层绘制橙色虚线包围框（`getClientRect` 通用计算，四种图形一致），
   随移动/删除/清空同步刷新；反馈层与对象登记表隔离的设计（PR #1）在此兑现。
5. **规则解析扩展**（rules.ts）：
   - 新意图：删除（删掉/去掉/移除…）、选中（选中/选择…）、移动（移/挪/放到…）；
   - 目标提取 `extractTarget`：特征优先；"刚才/上一个"→last；
     仅当无特征时代词（它/这个/那个）才解释为 selected；
   - 移动细节：步长词（一点→small/很多→large/中文数字→像素），<5 的数值视为
     "一/两"量词残留忽略；
   - **意图判定顺序**：清空 > 删除 > 选中 > 绘制（要求动词+图形词同现）> 移动。
     关键消歧："画一个圆放在左上角"是绘制（有画+圆），"把圆放到左上角"是移动（无绘制动词）。

## 验证方式

```
npx tsx scripts/parser-smoke.ts   # 22 用例全过（新增 10 个本 PR 用例）
npm run dev
```

手工链路（调试框依次输入）：

1. "画一个红色的圆" → "画一个蓝色的圆" → "选中那个红色的圆" → 红圆出现橙色虚线高亮；
2. "往右移一点" → 高亮的红圆右移，高亮框跟随；
3. "把它删掉" → 红圆消失，反馈"已删除圆形"；
4. "画两次圆后选中圆" → 两个圆都高亮，反馈"找到 2 个匹配对象…请说得更具体些"。

`npm run build`（含类型检查）通过。

## 已知限制

- "把它移到那个圆的右边"这类**相对锚点定位**会被误解析为"移到画布右侧"
  （`relativeTo` 是 P1 能力，届时由 LLM 通道产出，执行层增加 anchors 解析）；
- 序数/空间排序指代（"左边那个"/"第二个"）按设计收窄至 P1，当前会落入特征匹配或 miss。

## 追加修复（同分支第二次提交）

针对试用反馈做的三件事：

1. **端到端验证基建**：`scripts/e2e-repro.mjs`（puppeteer-core 驱动本机 Edge，
   走真实调试输入链路，检视 Konva 主层/反馈层内部状态并截图）；stage.ts 增加
   开发态调试钩子 `window.__sonink`。用它复核了"选中红色的会选成蓝色"的反馈——
   **当前分支无法复现**（选中、高亮、删除全部正确），现象与旧分支（无 select 意图）
   行为一致，怀疑试用时跑的是 feat/05 或未刷新页面。
2. **中文输入法回车误触**：调试输入从 `keyup.enter` 改为 `keydown.enter` +
   `isComposing` 守卫，输入法确认候选词的回车不再触发提交（误触会导致指令
   被提前/重复提交，是"图形复制了一个"的可能来源）。
3. **连接词级复合指令**（容错强化的一部分提前实现）：按 然后/接着/和/以及/逗号
   拆分子句，全部子句解析成功才采纳，任一失败回退整句解析（保证不回归）；
   无动词子句承接前句绘制意图。"画一个红色的圆和一个蓝色的圆"现在画两个；
   绘制动词新增"生成/创建"。冒烟用例扩至 26 个全过。
